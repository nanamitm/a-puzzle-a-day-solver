A-Puzzle-A-Day Solver

This is an unofficial solver of DragonFjord's A-Puzzle-A-Day and similar puzzles. You can find a solution for the selected date.

Note: If you use this tool, you may lose the joy of solving puzzles on your own. Be careful!

Options

Weekday is used only for "WeekDay Calendar Puzzle".

Tip You can prefill by URL. Example: ?date=2026-01-01&type=dragonfjord. Full help: README
